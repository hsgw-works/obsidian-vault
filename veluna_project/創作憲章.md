# 1. 因果の納得感

- 目的：物語の出来事に必然性を与え、読者を納得させる。
    
- 原則
    
    1. すべての主要な出来事に「原因」と「結果」を設定する。
        
    2. 原因は世界観やキャラ心理の中で一貫して説明できること。
        
    3. 情報は小出しにし、読者が自然に因果を発見できる形にする。
        
    
- チェック質問
    
    - この展開は「なぜそうなったか」を即答できるか？
        
    - 逆に「なぜそうならなかったか」も説明できるか？

# 2. 本質と現実の分離

- 目的：世界の形而上の法則（本質）と、それが現象化した日常（現実）を明確に区別する。
    
- 原則
    
    1. 本質は哲学・根源法則・理論。直接的には見えない。
        
    2. 現実はその本質の影響で現れる現象や出来事。
        
    3. 本質と現実はリンク可能にし、いつでも相互参照できる構造にする。
        
    
- チェック質問
    
    - この出来事の背後にある本質は何か？
        
    - その本質が現実にどう現れているかを描けているか？

# 3. キャラクターの孤立

- 目的：キャラクターを物語の都合ではなく、固有の意思で動かす。
    
- 原則
    
    1. キャラごとに明確な行動心理・目的・価値観を設定する。
        
    2. 世界の出来事や他キャラとの関わりは偶発的であってもよい。
        
    3. 他キャラとの噛み合わなさも物語のリアリティとして活かす。
        
    
- チェック質問
    
    - この行動は物語のためではなく、そのキャラ自身のために選ばれているか？
        
    - 他キャラとの関係性は、その時点での心理から自然に説明できるか？
